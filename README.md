# TextileFLow-Application-repo
This is the Repository of our application named "TextileFlow" which is build to facilitate the textile and garment merchants.The application contains many features such as payment tracking , Debtors management , Cashflow management , automatic invoice generation. The important feature of our app is that it can send reminders on Whatsapp to debtors regarrding there dues.
URL for the application : https://textile-working-capi-q6tv.bolt.host.
I have also uploaded the APK file for the application which is fully functional on the android mobile phone. You can find the Link to download the APK file in the Release Section of this repo
