# Paycast-Application-repo
This is the Repository of our application named "Paycast" which is build to facilitate the textile and garment merchants.The application contains many features such as payment tracking , Debtors management , Cashflow management , automatic invoice generation. The important feature of our app is that it can send reminders on Whatsapp to debtors regarrding there dues.
